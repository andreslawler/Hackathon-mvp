# Integrated Core Solution — Offer Structure (Bid Office Template)

A complete ICS offer is built from a defined set of aligned pre-sales assets. They share the same Work Package scope, so the Solution Description, Statement of Work, Work Breakdown Structure, Effort Estimates and Time Plan all describe the same solution. Any change in scope cascades into cost and time.

## The ICS offer asset set

| Asset | Purpose in the offer |
|---|---|
| Technical Solution Description (Appendix 3) | The end-to-end technical solution: components, architecture, deployment options, and work packages. |
| Statement of Work (Appendix 1) | Scope of services, in and out of scope, delivery approach, entry and exit criteria, acceptance criteria, RACI reference. |
| Work Breakdown Structure (WBS) | The delivery work packages, aligned to the SoW and Solution Description. |
| Detailed Time Plan | Project duration and milestones for the defined scope. |
| Effort Estimates | Services cost estimation for the defined scope. |
| Responsibility Matrix (RASCI) | Task-level responsibility split across Ericsson, Customer or Prime Integrator, and the 3PP CaaS supplier. |
| Risk Register | Project risk qualification and quantification, feeding Premium Proposal. |

## Construction principles

- **The scope is prescriptive.** Costing is based on the pre-defined SoW. Any deviation from the standard scope must be re-estimated, because it changes the Effort Estimates and the Time Plan.
- **Standard scope:** 4G/5G dual-mode core, no exposure use case, dimensioned for up to 1M SAU and roughly 120 Gbps on Ericsson CNIS, greenfield, geo-redundant 1+1, with a defined scaling path from 100K to 1M.
- **CNIS cloud infrastructure is mandatory** and included. 5GCAS and EVNFM are mandatory.
- **Optional packages** (4G and 5G roaming, MCPTT, network slicing, external HSS) are scoped and priced separately.

## Commercial construct

- Software licensing is capacity-based (by SAU and throughput), with one-time and recurring components.
- Services are estimated from the Effort Estimates for the defined work packages.
- Reach for the structured levers before any list-price move: multi-year framework, software-attach on optional packages, capacity-tier pricing, and rebate tied to deployment milestones.
- Present indicative pricing in ranges. Final terms are subject to detailed scope and formal commercial discussion.
